# Keystore Maker 0.0.1

Initial open-source release.

## Features

- Generate Android JKS keystores as `.jks` or `.keystore`.
- Generate PKCS12 keystores as `.p12` or `.pfx`.
- Inspect signature, MD5, SHA1, and SHA256 fingerprints.
- Chinese and English UI.
- Packaged runtime does not require Java, JDK, Android Studio, keytool, or Node.js.

## Downloads

- macOS Apple Silicon: `Keystore Maker-0.0.1-arm64.dmg`
- macOS Apple Silicon zip: `Keystore Maker-0.0.1-arm64-mac.zip`
- Windows x64: build with `npm run dist:win` on Windows or CI, then upload the generated installer/portable assets.

## Notes

The macOS build is not notarized. On first launch, macOS may require opening it from Finder with Control-click → Open.
